## My process

### Built with

- Semantic HTML5 markup
- CSS custom styling
- Flexbox (page alignment/layout)
- Responsive design with a mobile media query
- Custom typography using Google Fonts (`Outfit` and `Young Serif`)

### What I learned

While building this project, I practiced:

- Structuring content sections clearly (`Preparation time`, `Ingredients`, `Instructions`, `Nutrition`)
- Styling lists and section separators to match a design
- Creating a clean nutrition table with consistent row spacing and borders
- Handling responsive behavior, especially:
  - making the card layout adapt to smaller screens
  - removing border radius/shadow on mobile for full-screen layout behavior
  - understanding how container padding affects child width (like the recipe image)

### Continued development

In future projects, I want to keep improving:

- Mobile-first CSS workflow
- More precise responsive spacing and typography scaling
- Accessibility details (heading hierarchy, table semantics, and color contrast)

### AI collaboration

I used AI (GitHub Copilot) as a learning assistant during this project for:

- Debugging CSS layout issues
- Understanding why specific styling behavior happened (for example, image width and container padding)
- Getting guided suggestions for responsive improvements

AI was most helpful when I used it to understand concepts step by step, then implemented and tested changes myself.
